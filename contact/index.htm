Contact

If you ever have any questions or if you want to report an error, just write us an email. Please be sure to write the email in English. We will only answer emails written in English.

If you have a question, please check if it is not already listed on the FAQ page.

If you get an error, please try to describe it as much as possible. It is hard for us to figure out what kind of error you had if you just write „not working” or „error”. If you see any error codes or error messages, add them to your message. Also check our Errors page for error details before reporting your error. Thanks.
